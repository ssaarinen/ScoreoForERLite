# ScoreO / Rogaining extension for EResults Lite

This project is an extension for [EResults Lite (v4)](https://www.oriento.fi/eresults/lite) that adds support for Score-O
or Rogaining events. Extension reads competition file from EResults Lite and processes it according to Score-O and Rogaining 
rules. The extension can be used to view results in real time while results are being received via EResults Lite or 
to generate the final results after the event.

Like EResults Lite, this extension is targeted to recreational events and small "competitions". For larger events
or more advanced features, consider using specialized software designed for these event types.

# ScoreO / Rogaining -laajennus EResults Lite:lle

Tämä projekti on laajennus [EResults Lite (v4)](https://www.oriento.fi/eresults/lite) -sovellukselle, joka lisää tuen
Score-O- ja Rogaining-tyyppisten tapahtumien tulosten käsittelyyn. Laajennus lukee EResults Liten kilpailutiedostoa
ja käsittelee sen Score-O- ja Rogaining-sääntöjen mukaisesti. Sitä voi käyttää tulosten reaaliaikaiseen tarkasteluun, kun
EResults Lite vastaanottaa tuloksia, sekä lopullisten tulosten luomiseen tapahtuman jälkeen.

Kuten EResults Lite, tämä laajennus on suunnattu harrastetapahtumiin ja pieniin "kilpailuihin". Suurempia tapahtumia
tai edistyneempiä ominaisuuksia varten kannattaa harkita näihin tapahtumatyyppeihin suunniteltuja erikoisohjelmistoja.
